🔗 API 获取指南
========================================================================

1️⃣  Tushare Pro Token
   URL: https://tushare.pro
   - 点击"注册"
   - 完成实名认证（免费）
   - 在"个人中心"获取 Token
   - 复制 Token 到 .env 文件

2️⃣  OpenAI API Key
   URL: https://platform.openai.com/account/api-keys
   - 创建账户
   - 充值费用（从 $5 起）
   - 生成新的 API Key
   - 复制 Key 到 .env 文件

💡 成本参考（分析 20 只股票/天）:
   • 使用 gpt-3.5-turbo: ~¥2.20/天
   • 使用 gpt-4-turbo: ~¥14.6/天
   • 使用 gpt-4: ~¥30+/天
