🔑 核心功能一览
================================================================================

✅ 数据获取
   • 自动从 Tushare Pro API 下载 A 股日线数据
   • 支持单个或批量下载
   • 自动错误处理和重试机制
   • 返回标准化的 pandas DataFrame

✅ 技术分析
   • 计算移动平均线 (MA5, MA10, MA20)
   • 计算成交量指标 (VOL_MA, VOLUME_RATIO)
   • 计算日涨跌幅和 5 日高低点
   • 完整的特征工程和向量构建

✅ 智能分析
   • 调用 OpenAI API（支持 GPT-4、GPT-4-turbo、GPT-3.5-turbo）
   • 基于技术面生成交易信号
   • 详细的分析理由说明

✅ 交易信号
   生成：BUY / WATCH / SELL
   包含：
   • score (0-100) 综合评分
   • confidence (0-100) 信心指数
   • entry_price 建议入场价
   • stop_loss 止损价格
   • target_price 目标价格
   • reason 详细理由

✅ 报告生成
   • 输出 CSV 格式报告
   • 支持标准版和详细版
   • 自动按评分降序排列
   • 自动保存到 report/ 目录

🚀 快速开始（3步，5分钟）
================================================================================

第 1 步：配置 API Keys
─────────────────────────────────────────────────────────────────────────────
  a) 获取 Tushare Token
     访问 https://tushare.pro → 注册 → 获取 Token
  
  b) 获取 OpenAI API Key
     访问 https://platform.openai.com → API Keys → 生成新 Key
  
  c) 配置 .env 文件
     $ cp .env.example .env
     $ vim .env (修改 TUSHARE_TOKEN 和 OPENAI_API_KEY)

第 2 步：安装依赖
─────────────────────────────────────────────────────────────────────────────
  $ pip install -r requirements.txt

第 3 步：运行程序
─────────────────────────────────────────────────────────────────────────────
  选项 A（推荐）：使用启动脚本
    $ bash run.sh                  # macOS/Linux
    $ run.bat                      # Windows
  
  选项 B：直接运行主程序
    $ python ai_stock_agent/main.py
  
  选项 C：运行示例程序
    $ python example_usage.py
